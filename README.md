# Air Quality Index Analysis

Machine learning system to predict AQI levels across major Indian cities
(Delhi, Bangalore, Kolkata, Hyderabad) using pollutant data.

## Tech Stack
- ML: Scikit-learn, XGBoost, LightGBM, SMOTE
- Backend: FastAPI + Docker
- Frontend: React + Tailwind CSS
- Deployment: Render / Hugging Face Spaces + Vercel

## Project Phases
- Phase 1: EDA
- Phase 2: Preprocessing + SMOTE
- Phase 3: Model Training
- Phase 4: Model Evaluation
- Phase 5: FastAPI Backend
- Phase 6: React Frontend
- Phase 7: Deployment
